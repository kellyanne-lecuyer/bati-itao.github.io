Conception avec du contenu généré par une CSS

Une CSS peut générer du texte au moyen des pseudo-classes « :before » et « :after », mais le texte ne figure pas dans le modèle DOM et n’est donc pas facilement accessible aux technologies d’assistance. Bien que les lecteurs d’écran NVDA et VoiceOver reconnaissent le contenu produit par une CSS, ce n’est pas le cas des lecteurs d’écran JAWS et Narrator.

Par conséquent, on peut utiliser un contenu important généré par une CSS sur des feuilles de style d’impression pour imprimer l’adresse URL des liens sur la page, mais autrement il faut l’éviter.

Si vous utilisez du contenu produit par une CSS, masquez-le pour les lecteurs d’écran au moyen de l’attribut aria-hidden=”true”. Si le contenu généré par une CSS est informatif, fournissez un équivalent textuel. Pour ce faire :

  • Si le contenu est un lien ou une commande, utilisez l’attribut « aria-label ».
  • Si le contenu ne peut être ciblé, utilisez la classe « .wb-inv » de la BOEW pour masquer visuellement l’équivalent textuel.

 Bon exemple : Masquer visuellement le contenu généré par une CSS et fournir un équivalent textuel

Dans cet exemple, l’icône de marteau Font Awesome est utilisée pour représenter une tâche. L’élément <span> avec la classe fas fa-hammer affiche l’icône par l’entremise d’une pseudo-classe de CSS, tandis que son attribut title contient une infobulle pour les utilisateurs de souris et que l’attribut aria-hidden="true" cache l’élément aux lecteurs d’écran. Lle nom « Tâche », masqué visuellement, est indiqué juste après et est visible pour les lecteurs d’écran.

Code commence

<style type="text/css">
	.fa-hammer:before { content: "\f6e3"; }
	.fa, .far, .fas {  font-family: "Font Awesome 5 Free"; }
	li.icon { list-style: none; }
	li.icon > * { margin-left: 2em; }
	.wb-inv {
	   clip: rect(1px,1px,1px,1px);
	   height: 1px;
	   margin: 0;
	   overflow: hidden;
	   position: absolute;
	   width: 1px;
	}
</style>

Fins du code

Code commence

<ul>
   <li class="icon">
      <span class="fas fa-hammer" title="Tâche" aria-hidden="true"></span>
      <span class="wb-inv">Tâche</span>
      Remplir la demande
    </li>
 </ul>
